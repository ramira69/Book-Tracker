# Book-Tracker
GUI-приложение «Book Tracker» для управления списком прочитанных книг с возможностью фильтрации


# Book Tracker — Трекер прочитанных книг

## Автор: Осокин Максим Сергеевич
## Описание программы:
Приложение для учёта прочитанных книг с графическим интерфейсом. Позволяет добавлять книги с указанием названия, автора, жанра и количества страниц. Есть фильтрация по жанру и количеству страниц. Данные сохраняются в файл `books.json`.

## Инструкция по запуску:
1. Установите Python 3.x (если не установлен).
2. Установите необходимые библиотеки:
   ```
   pip install -r requirements.txt  # (если потребуется в будущем)
   ```
3. Запустите приложение:
   ```
   python book_tracker.py
   ```
4. Используйте интерфейс для добавления и фильтрации книг.
5. Данные сохраняются автоматически при добавлении книги.
6. Для загрузки данных при следующем запуске просто откройте программу.

## Примеры использования:
- Добавьте книгу: введите все поля и нажмите «Добавить книгу».
- Фильтрация: введите жанр (например, «фантастика») или количество страниц (например, «200») и нажмите «Применить фильтр».
- Данные хранятся в `books.json` и загружаются при старте.
